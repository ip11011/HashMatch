# SHA-256 Hash Match Tool 🔐

Check a SHA‑256 hash against your list of passwords. Works in Python or directly in a web browser.

---

## 💻 Terminal Version

### Language

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge\&logo=python\&logoColor=white)

### How it works

1. Reads passwords from `passwords.txt`
2. Asks for a SHA‑256 hash
3. Hashes each password and compares
4. Shows the original password if a match is found
5. Prints “No matching passwords” if none match

### How to use

1. Add your passwords to `passwords.txt` (one per line)
2. Run `HashMatch.py`
3. Enter the hash to check
4. Press Enter to exit

3. Click **Check Hash**
4. See the original password if a match is found

### How to use

* Open [`web.html`](https://ip11011.github.io/HashMatch/web.html) in a browser
* Enter hash and passwords
* Click the button → result appears instantly
